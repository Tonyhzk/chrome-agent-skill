# Set up extension

## Get Started

1. Install the [Browser MCP extension](https://browsermcp.io/install).
2. Pin the Browser MCP extension for easy access.
   ![Pin the Browser MCP extension](https://mintcdn.com/browsermcp/e19Ahc63CA2bWPva/images/extension/pin-instructions.png?fit=max&auto=format&n=e19Ahc63CA2bWPva&q=85&s=549374b22221c097148085d670679396)
3. Open the Browser MCP extension by clicking the extension icon in the toolbar.
4. Click "Connect" to connect the current tab to the MCP server.
   ![Connect to the MCP server](https://mintcdn.com/browsermcp/e19Ahc63CA2bWPva/images/extension/connect-extension.png?fit=max&auto=format&n=e19Ahc63CA2bWPva&q=85&s=b1c78da850f219b0f913bf84c92aafbf)
   All browser actions will be performed on the connected tab.
