# Welcome

Welcome to [Browser MCP](https://browsermcp.io/), the MCP server for automating your browser.

## What is MCP?

[Model Context Protocol (MCP)](https://modelcontextprotocol.io/) allows AI applications to interact with other applications and services.
With Browser MCP, you can use MCP to automate your browser so that AI applications can navigate the web, fill out forms, and more.
